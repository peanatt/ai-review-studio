📚 คลังข้อความสำเร็จรูป
เลือกหมวดสินค้า แล้วเลือกข้อความที่ชอบ — ใช้ได้ฟรี ไม่ต้องมี API Key
📖 คู่มือการใช้งาน
🖼️ 1. เริ่มต้นสร้างภาพ
แตะ ➕ หน้าเปล่า ในเมนู "เทมเพลต" เพื่อเริ่มจากศูนย์ หรือเลือกเทมเพลตสำเร็จรูปที่ชอบ แล้วแนบรูปสินค้าของคุณ
📚 2. คลังข้อความสำเร็จรูป (ฟรี ไม่ต้องมี API Key)
อยู่ตรงกึ่งกลางหน้าจอ ใต้รูปตัวอย่าง — มองหาแถบ "✏️ แก้ไขข้อความในภาพ" แล้วแตะลิงก์ "📚 คลังข้อความสำเร็จรูป" ที่มุมขวาของแถบนั้น จะมี 8 หมวดสินค้า × 6 ภาษา ให้เลือกใส่ในภาพได้ทันที
💬 3. คลังแคปชั่นสำเร็จรูป
อยู่ในแผงด้านขวา แท็บ "✨ AI สร้าง" (แท็บแรกที่เปิดมาเจอ) — มองหาหัวข้อ "แคปชั่น / Content" แล้วแตะ "📚 คลังแคปชั่น" สีฟ้า ข้างปุ่ม "🔄 สร้างใหม่"
✨ 4. ให้ AI ช่วยเขียน (ถ้าใส่ API Key)
ถ้าต้องการให้ Claude เขียนแคปชั่นสดใหม่ทุกครั้งตามภาษา/แพลตฟอร์มที่เลือก ให้ใส่ API Key ของคุณในเมนู "⚙️ ตั้งค่า" — ถ้าไม่ใส่ ระบบจะใช้ตัวสร้างข้อความออฟไลน์หรือคลังสำเร็จรูปแทน (ไม่มีค่าใช้จ่าย)
🎨 5. แต่งภาพเพิ่มเติม
ใช้แท็บ ข้อความ / ฟิลเตอร์ / สไตล์ / เลเยอร์ ในแผงด้านขวา เพื่อปรับฟอนต์ สี สติ๊กเกอร์ เวคเตอร์ และแพทเทิร์นพื้นหลัง
📥 6. บันทึก/ดาวน์โหลด
เมื่อพร้อมแล้ว ใช้ปุ่มดาวน์โหลด/ส่งออกที่แถบเครื่องมือด้านบนของรูปเพื่อบันทึกภาพ
◆ เพิ่มเวคเตอร์
เลือกรูปทรง แล้วปรับสีได้ก่อนเพิ่ม — ลากเพื่อจัดตำแหน่งบนแคนวาสได้ทีหลัง
